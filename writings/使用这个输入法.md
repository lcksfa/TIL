現在總算是在使用小鶴雙拼了，只是 還需要把 它變成 簡體字才行。

修改 小鹤双拼的 配置文件 如下：

```cpp
switches:
  - name: ascii_mode
    reset: 0
    states: [ 中文, 西文 ]
  - name: full_shape
    states: [ 半角, 全角 ]
  - name: simplification
    reset: 1
    states: [ 漢字, 汉字 ]
  - name: ascii_punct
    states: [ 。，, ．， ]
```

事实上 ，改变的只有 name: simplification 下一行的 reset 的 1 而已；
